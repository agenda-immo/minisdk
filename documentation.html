Notice d'intégration boutons de « prise de rendez-vous en ligne »  Estimation, Bien spécifique et Négociateur Spécifique et agence

À destination du webmaster du site immobilier.

Agenda Immo est une solution 100% web permettant aux agences immobilières d'offrir la prise de rendez-vous en ligne sur leur site agence. Nous offrons 4 types prise de rendez-vous en ligne : 

  1. Bouton RENDEZ-VOUS ESTIMATION
  2. Bouton RENDEZ-VOUS VISITE POUR UN BIEN SPÉCIFIQUE
  3. Bouton RENDEZ-VOUS AVEC NÉGOCIATEUR SPÉCIFIQUE
  4. Bouton RENDEZ-VOUS AGENCE

Pour ajouter Agenda Immo à votre site

Intégrez le code suivant pour beneficier de tous les boutons.

Dans la section Head de votre site, ajoutez le code suivant :

<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/agenda-immo/minisdk@main/sdk.min.js"></script>

1 - Bouton de prise de rendez-vous Estimation :

Le bouton « rendez-vous estimation », à destination de la page estimation ou gestion de biens locatifs pour le prospect vendeur ou bailleur.

À destination de :

  • la page estimation vente (vendeur) et qui redirigera vers l'url : https://pro.agendaimmo.com/rdvs/estimation/nom-agence/vente

  • la page estimation location (bailleur) et qui redirigera vers l'url : https://pro.agendaimmo.com/rdvs/estimation/nom-agence/location

  • ou/et d'une landing page demandant au client si il désire une estimation pour un bien en vente ou en location ( ex page estimation générale et page d'accueil de votre site )  redirigeant vers l'url : hhttps://pro.agendaimmo.com/rdvs/estimation/nom-agence

La variable : nom-agence se trouve dans la fiche agence qui vous a été transmise après votre inscription.

Code d'intégration

<script>
agendaimmo_estimation("votre lien estimation")
</script>

2 - Bouton de prise de rendez-vous pour visiter un bien spécifique :

Ce bouton   est à destination de la page bien concernant la vente ou la location pour le prospect acquéreur ou locataire. Ce lien permet d'attribuer le rendez-vous à/aux négociateur(s) en charge du bien tout en prenant compte de leurs disponibilités ainsi que de celles du bien (vendeur ou résident).

Code d'intégration

<script type="text/javascript">agendaimmo(agence_id,reference_interne,options);</script> Paramètres
  • agence_id: est l'identifiant qui vous a été fourni lors de votre inscription
  • reference_interne: est la référence passée lors de l'import du bien via le webservice AgendaImmo.
  • options: vous permet de modifier l'affichage

3- Bouton de prise de rendez-vous avec un négociateur spécifique

Code d'intégration

<script type="text/javascript">
const options = {
   id: "button_nego",
   styles_container : "background:transparent;",
   styles_button: "color:#000;text-transform:none;text-decoration:none;text-align: left;font-size: 16px;font-family: Arial;font-weight: bolder;",
   text: "Prendre rendez-vous",
   theme_logo: "light"
};

agendaimmo_negociateur(agence_id,email ou n° de téléphone,options);
</script> Paramètres
  • agence_id: est l'identifiant qui vous a été fourni lors de votre inscription
  • adresse email ou numero de telephone: l'adresse email ou le numéro de téléphone renseigné(e) dans le profil AgendaImmo du négociateur
  • options: vous permet de modifier l'affichage

4- Bouton prise de rendez-vous Agence

Le bouton  « Rendez-vous agence»   permet de prendre rendez-vous avec l’un des membres de l’agence au choix. (Page de destination à valider avec le client).

Code d'intégration

<script type="text/javascript">agendaimmo_agence(agence_id, options);</script> Paramètres
  • agence_id: est l'identifiant qui vous a été fourni lors de votre inscription
  • options: vous permet de modifier l'affichage

Options

L'affichage du bouton peut être configuré au travers d'un objet d'options passé lors de l'intialisation.

Les options définies par défaut sont les suivantes :


const default_options = {
styles_container: "position:absolute;width: 380px;height:100px;top:75px;right:0px;z-index:999",
styles_button:
"border-radius:60px 0px 0px 60px;text-decoration:none;padding: 0 15px;text-align: center;display: flex;align-items: center;justify-content: space-around;font-size: 25px;font-family: Arial;font-weight: bolder;", styles_icon: "width: 28%; height: auto;",
text: "rendez-vous visite",
theme_logo: "light",
};

Exemples

Les exemples suivants s'appliquent à tous les boutons.

Mode sombre

<script type="text/javascript">
agendaimmo(agence_id,reference_interne,{ theme_logo: "dark"})
</script>

Changer la couleur du bouton

<script type="text/javascript">
agendaimmo(agence_id,reference_interne,{styles_container: {background: '#183c63'}})
</script>

Modifier l'affichage

<script type="text/javascript">
agendaimmo(agence_id,reference_interne,{ text: "Mon texte personnalisé",
styles_container: { borderRadius: "5px", padding: "5px" },
styles_button: { textTransform: "lowercase" },
})
</script>
